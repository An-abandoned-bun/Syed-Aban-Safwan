## Featured Projects

### PiezoPower Analyzer

A Python-based dashboard for analyzing voltage data generated by piezoelectric tiles.

**Key features:**

* Detects footsteps from voltage spikes
* Calculates peak and average voltage
* Estimates capacitor energy storage
* Visualizes voltage data using graphs
* Exports experiment reports

**Skills used:** Python, Streamlit, Pandas, Matplotlib, data analysis, electrical engineering concepts

---

### Piezoelectric Tile Energy Monitor

An Arduino-based prototype that measures electricity generated from footsteps on piezoelectric tiles.

**Key features:**

* Reads voltage from piezoelectric tiles
* Counts total steps
* Lights an LED with each step
* Estimates energy stored in a capacitor
* Demonstrates a basic energy-harvesting concept

**Skills used:** Arduino, sensors, circuit design, capacitor energy calculation, embedded systems
